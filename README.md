# DJDX PEAK Studio

Native macOS tool for building and labeling IIDX result screens, and exporting the final model using Core ML. Uses Swift, SwiftUI, Vision framework, Core Graphics, paired with Python scripts.

The models generated by this tool will be used in [DJDX](https://github.com/katagaki/DJDX).

## Build & run

```sh
./Scripts/bundle.sh
open "Outputs/DJDX PEAK Studio.app"
```

## Original inputs

Label bounding box coordinates and sizes are included with the repository. However, the original input photos are not included.
If you would like access to the original input photos, please open a GitHub issue with your email address and I will get in touch as soon as possible.
